# 클래스형 vs 함수형
| 구분 | [클래스형](./script-class.js)                         | [함수형](./script-function.js)                                   |
| ---- | -------------------------------- | ---------------------------------------- |
| 구조 | 객체지향, 상태와 메서드 묶음     | 로직 중심, 함수 단위 분리                |
| 장점 | 캡슐화, 상속, 재사용성 유리      | 테스트 용이, 코드 단순, 재사용 쉬움      |
| 단점 | this 바인딩 이슈, 복잡할 수 있음 | 전역 함수 남발 시 충돌, 상태 관리 어려움 |
